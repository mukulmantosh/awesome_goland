# Demo Walkthrough

### Hide all Tool Windows

![demo](./hide_all_tool_windows.gif)

You can hide all the tool windows in the IDE and quickly switch to the editor via <kbd>⌘⇧F12</kbd> (macOS) / <kbd>Ctrl+Shift+F12 (Windows/Linux)</kbd>
