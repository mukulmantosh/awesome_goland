# Demo Walkthrough

### Quickly Handle Errors in Go

![demo](./handle_error.gif)

Press <kbd>⌘⌥⏎</kbd> (macOS) / <kbd>Ctrl+Alt+Enter</kbd> (Windows/Linux) on a function call that returns an error and select **Handle error**.
