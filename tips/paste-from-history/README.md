# Demo Walkthrough

### Paste from History

![demo](./9.gif)

The _Paste from History_ feature you to use the (limited) history of everything you put in the clipboard.

Press <kbd>⌘⇧V</kbd> (macOS) / <kbd>Ctrl+Shift+V</kbd> (Windows/Linux) to open a pop-up window.
